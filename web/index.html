Global Soil Profile Explorer

A web interface for exploring the Global High-Resolution Soil Profile Database — providing DSSAT-compatible soil profiles for crop simulation modeling across 225 countries.

How to Use

  • Click anywhere on the map to query soil data at that location
  • Or search by address / coordinates in the right panel
  • Download results as a .SOL file (DSSAT-ready) or JSON
  • Toggle between Globe and Mercator projections using the map controls

Original Dataset

Global High-Resolution Soil Profile Database for Crop Modeling Applications

Coverage225 countries · ~2 million profiles
Resolution5 arc-minutes (~10 km grid)
Layers6 layers: 5, 15, 30, 60, 100, 200 cm depth
FormatDSSAT *.SOL (v4.51)
Published2015 · Harvard Dataverse, V2
DOI 10.7910/DVN/1PEEY0

Data Sources

Soil properties (bulk density, organic carbon, clay, silt, pH, CEC) are derived from SoilGrids 1km (ISRIC, 2014) and ISRIC-AfSIS. Hydraulic properties (field capacity, wilting point, saturated conductivity) are estimated using the Saxton & Rawls (2006) pedotransfer function. Additional parameters use the HarvestChoice HC27 reference database.
