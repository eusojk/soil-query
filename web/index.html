Global Soil Explorer beta

A lightweight web tool for exploring global soil profiles. It is built on top of an outstanding open dataset so that researchers can access DSSAT-compatible soil data without downloading gigabytes of files.

How to Use

  • Click anywhere on the map to query soil data at that location
  • Or search by address/coordinates in the right panel
  • Download results as a .SOL file (DSSAT-ready) or JSON
  • Toggle between Globe and Mercator projections in the lower left corner
  • To return to these instructions again, click on the info icon in the top right corner

Original Dataset

The underlying data comes from a landmark 2015 study which addressed a longstanding gap in gridded soil inputs for crop simulation models like DSSAT. By combining SoilGrids 1km and ISRIC-AfSIS, the authors produced soil profiles on a 5 arc-minute grid with key hydraulic properties estimated via pedotransfer functions.

Please cite the original authors if you use this data in your work:

Han, Eunjin; Ines, Amor; Koo, Jawoo, 2015. "Global High-Resolution Soil Profile Database for Crop Modeling Applications." Harvard Dataverse, V1.
http://dx.doi.org/10.7910/DVN/1PEEY0
